### Dynamically create the table

> difficulty: `medium`

Create a function that takes array of objects and builds the table.

#### Entry data

```javascript
const students = [
  {
    id: 1,
    name: 'Andrii',
    age: 22,
    isWorkExperience: true
  },
  {
    id: 2,
    name: 'Anna',
    age: 21,
    isWorkExperience: false
  },
  {
    id: 3,
    name: 'Marta',
    age: 23,
    isWorkExperience: true
  },
];
```

#### Result

|id|name|age|isWorkExperience|
|---|---|---|---|
|1|Andrii|21|yes|
|2|Anna|20|no|
|3|Marta|23|yes|
